# 手相鑑定サイト デザインバリエーション

静的HTMLとして4種類のデザインを用意しています。いずれも `assets/kansei.jpg` と `assets/images/tesoutop.jpg` を活用しています。

- `index.html` — クラシックで落ち着いた雰囲気の基本バージョン
- `version2.html` — 夜のリトリートをイメージしたモダンデザイン
- `version3.html` — 植物をモチーフにしたオーガニックスタイル
- `version4.html` — ミニマルでリッチなプレミアムデザイン

各ファイルをブラウザで直接開くとデザインを確認できます。
